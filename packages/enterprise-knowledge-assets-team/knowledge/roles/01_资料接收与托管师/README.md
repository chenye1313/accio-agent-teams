# 01 资料接收与托管师私有知识库

## 用途

本目录定义资料接收与托管师的操作标准：资料进入、来源编号、脱敏权限、完整度判断、四库路由和交接格式。

## 文件清单

| 文件 | 用途 |
|---|---|
| `01_资料接收状态机.md` | 判断资料入口阶段和阻断条件 |
| `02_来源索引与编号规则.md` | 建立 `asset_id`、`source_id` 和来源索引 |
| `03_脱敏与权限标注规则.md` | 处理敏感信息、公开权限和不可外发内容 |
| `04_资料完整度与四库路由规则.md` | 判断资料是否够用，以及进入哪类知识库 |
| `05_输出模板与交接格式.md` | 输出 `source_index`、`asset_manifest` 和交接包 |
