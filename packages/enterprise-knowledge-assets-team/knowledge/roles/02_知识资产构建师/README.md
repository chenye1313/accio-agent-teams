# 02 知识资产构建师私有知识库

## 用途

本目录定义知识资产构建师的操作标准：如何接收上游资料包，如何拆分资料覆盖信息，如何做术语实体统一，如何抽取企业、产品、问答、案例四库候选条目，以及如何沉淀组织实践标签。

## 文件清单

| 文件 | 用途 |
|---|---|
| `01_构建状态机.md` | 判断构建阶段和阻断条件 |
| `02_资料拆分覆盖清单.md` | 定义每类资料需要拆出的信息 |
| `03_术语实体标准化规则.md` | 统一产品、客户、认证、行业、证据实体 |
| `04_企业与产品信息抽取规则.md` | 企业信息库和产品信息库候选条目抽取 |
| `05_客户问答与案例抽取规则.md` | 客户问答库和客户案例库候选条目抽取 |
| `06_组织实践标签与方法内沉规则.md` | 方法经验内沉和组织实践标签 |
| `07_输出模板与交接格式.md` | 标准输出和交接给质检官的格式 |
