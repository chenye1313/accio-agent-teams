# 03 知识质量与分发官私有知识库

版本：v0.1  
适用角色：`03_知识质量与分发官`

## 定位

本知识库用于把 `02_知识资产构建师` 输出的四库候选条目，验收为可用知识、待验证知识、内部知识或拒绝知识，并按六组任务打包分发。

核心判断：

```text
没有来源，不放行。
没有权限，不外发。
没有场景，不分发。
没有回流，不算闭环。
```

## 质量口径

20 分制用于判断条目能否进入正式库：

| 维度 | 分值 | 说明 |
|---|---:|---|
| 准确性 | 0-5 | 是否忠于来源，没有编造 |
| 完整性 | 0-5 | 字段是否支撑四库最小结构 |
| 可追溯 | 0-5 | 关键结论是否能回到 `source_id` |
| 可调用 | 0-5 | 是否能被六组用于具体业务动作 |

低于 16 分不能进入正式库；低于 12 分必须返工或拒绝。
