# Knowledge Package

这个目录是 `外贸经营定位专家Team` 的交付知识包。

它不是源知识库的唯一副本，而是从上层源知识库中筛选出来、适合跟随 Team 一起传输 / 上传 / 交付的知识层。

## 目录

| 目录 | 用途 |
|---|---|
| `shared/` | Team 共享知识底座 |
| `roles/` | 每个 Agent 的私有知识切片 |
| `cases/` | 脱敏案例与演示样例 |
| `raw_refs/` | 原始材料引用，不默认放原文 |

## 规则

- 共享知识只放所有角色都需要的底座。
- 角色私有知识放到 `roles/<role_id>/`。
- 每个 Agent 自己的角色文件夹里放 `knowledge_refs.yaml`。
- 原始书籍、完整 PDF、客户后台数据默认不放进交付包。
