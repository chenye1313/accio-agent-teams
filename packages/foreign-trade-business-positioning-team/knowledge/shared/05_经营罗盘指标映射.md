# 05 经营罗盘指标映射

版本：v0.2

用途：经营罗盘不是经营定位的替代品，而是经营定位的验证系统。新版要求每份报告先建立上线前基线，再定义 7/30/90 天复盘指标。

经营定位回答：

```text
做哪盘生意、服务谁、凭什么赢
```

经营罗盘回答：

```text
这个定位有没有带来经营改善
```

## 1. 三大指标维度

### 1.1 营销效能：钱花得对不对

| 指标 | 定义 | 经营定位相关判断 |
|---|---|---|
| 曝光量 | 产品或店铺被看到的次数 | 关键词、类目、投放入口是否足够 |
| 点击量 / CTR | 展示后获得点击的数量和比例 | 首屏定位、关键词、产品图片是否吸引目标买家 |
| CPL 单询盘成本 | 获得一个询盘的平均成本 | 目标买家是否精准，投放是否浪费 |
| 平均点击成本 | 单次点击成本 | 关键词竞争是否过热，投放是否需要收窄 |
| 综合 ROI | 投入产出比 | 定位、投放、转化是否形成正循环 |

### 1.2 团队转化：人能不能转成订单

| 指标 | 定义 | 经营定位相关判断 |
|---|---|---|
| 访客转化效率 | 访客转询盘的效率 | 页面表达、信任资产、CTA 是否到位 |
| 询盘转化效率 | 询盘转报价 / 深沟通的效率 | 业务回复是否贴合买家 Job |
| 报价订单率 | 报价后形成订单的比例 | 定价、差异化、谈判话术是否有效 |
| 首响时间 | 第一次回复买家的速度 | 转化差是否由业务响应造成 |
| 有效跟进率 | 有价值询盘被持续跟进的比例 | 高价值买家是否漏接 |

### 1.3 经营健康：货和客户结构是否健康

| 指标 | 定义 | 经营定位相关判断 |
|---|---|---|
| 有效询盘率 | 有价值询盘占比 | 目标买家与表达是否匹配 |
| 老客户复购比 | 复购订单占比 | 产品价值、服务体验、客户生命周期是否健康 |
| 爆款集中度 | 销售集中在少数爆品的程度 | 产品结构机会与风险 |
| 国家 / 市场集中度 | 询盘或订单集中在哪些国家 | 市场定位是否过散或过窄 |
| SKU 利润结构 | 不同产品的毛利与成交贡献 | 主推产品是否真的赚钱 |

## 2. 上线前基线

任何页面改版、定位调整、关键词调整前，MetricQualityGate 必须尽量记录基线。

| 基线项 | 最小周期 | 用途 |
|---|---|---|
| 曝光、点击、CTR | 近 30 天 | 判断流量入口和首屏吸引力 |
| 访客、PV、访客转询盘 | 近 30 天 | 判断页面转化 |
| 询盘数、有效询盘率 | 近 30/90 天 | 判断目标买家是否精准 |
| Top 询盘产品 | 近 30/90 天 | 判断哪些 SKU 真带商机 |
| Top 成交产品 | 近 90 天 | 判断哪些 SKU 真带收入 |
| 国家分布 | 近 90 天 | 判断目标市场是否真实 |
| 广告花费、平均点击成本 | 近 30 天 | 判断投放效率 |
| 首响时间、报价转化 | 近 30 天 | 判断业务响应和报价质量 |

如果没有基线，报告可以给行动建议，但不能声明“预计提升多少”。

## 3. 7/30/90 天复盘节奏

### 7 天：上线可用性检查

关注：

- 新首屏是否正常显示。
- CTA 是否能收到询盘或附件。
- 业务员是否理解新定位和买家筛选标准。
- 是否出现明显错误流量或误解。

输出：

- `first_7_day_check`
- 需要立即修正的页面、关键词、表单或话术。

### 30 天：先导指标复盘

关注：

- CTR 是否变化。
- 访客转询盘是否变化。
- 有效询盘率是否变化。
- Top 询盘产品是否更接近主推产品。
- 询盘里是否出现更明确的目标买家词。

输出：

- `first_30_day_metric`
- 继续放大、微调或回滚建议。

### 90 天：经营结果复盘

关注：

- 报价订单率。
- 客单价或订单金额区间。
- 高价值买家占比。
- 复购或二次询盘。
- 广告 ROI 和市场结构。

输出：

- `first_90_day_metric`
- 下一轮定位、产品、投放或供应链动作。

## 4. 经营问题诊断分流

| 现象 | 优先判断 | 退回角色 |
|---|---|---|
| 曝光低 | 关键词、类目、投放覆盖不足 | MetricQualityGate / Expression |
| CTR 低 | 首屏定位、标题、主图不吸引目标买家 | Expression / Differentiation |
| 访客多但询盘少 | 页面信任不足、CTA 不清楚、产品结构不匹配 | Expression / SupplyMapping |
| 询盘多但低质 | 买家定位、关键词、人群入口错位 | BuyerJTBD / Differentiation |
| 询盘准但报价少 | 业务响应、资料包、报价流程不足 | MetricQualityGate / BusinessModel |
| 报价多但订单少 | 价格逻辑、证据资产、交付承诺不足 | BusinessModel / SupplyMapping |
| 成交产品与主推不一致 | 老板想推和市场真实需求错位 | BusinessModel |
| 高询盘产品低毛利 | 生意模式和产品结构需要重排 | BusinessModel / Differentiation |

## 5. 经营定位动作与指标映射

| 定位动作 | 预期影响指标 | 验证周期 | 若无改善，优先复盘 |
|---|---|---|---|
| 明确 A 类买家 | 有效询盘率 / CPL | 30 天 | 目标买家 / 关键词 / 页面首屏 |
| 重写首屏定位 | CTR / 访客转化效率 | 14-30 天 | Slogan / 主图 / 信任状 |
| 增加信任资产 | 询盘转化效率 / 报价订单率 | 30-60 天 | 案例、认证、交付过程 |
| 优化差异化报价逻辑 | 报价订单率 / 综合 ROI | 30-60 天 | 报价模板、谈判话术 |
| 调整主推 SKU | Top 询盘产品 / Top 成交产品 / 毛利结构 | 30-90 天 | 产品结构、价格带、页面入口 |
| 设计老客激活动作 | 老客户复购比 | 60-90 天 | 客户分层、产品推荐 |

## 6. MetricQualityGate 输出要求

每个定位报告必须声明：

1. 当前定位优先改善哪个指标。
2. 上线前有哪些基线，哪些基线缺失。
3. 7 天内检查什么可用性问题。
4. 30 天内看哪个先导指标。
5. 60-90 天内看哪个结果指标。
6. 如果指标不动，优先复盘哪一层。

## 7. 红线

如果定位动作不能绑定至少 2 个经营罗盘指标，则不能通过最终质量门。

如果没有上线前基线，不能承诺提升比例，只能写“建议观察指标”。
