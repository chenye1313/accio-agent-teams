# 01 能力证据分级规则

版本：v0.1  
适用角色：`04_SupplyMapping供给能力映射师`

## 1. 核心判断

SupplyMapping 不负责创造优势，只负责验证优势。

每条优势都必须回答：

```text
它服务哪个买家 Job？
它改善哪个 Desired Outcome？
它有什么证据？
它是不是同行通用卖点？
```

## 2. 能力类型

| 能力类型 | 常见字段 | 示例 |
|---|---|---|
| 产品能力 | `main_products`、`product_differentiation` | 产品组合、特殊材质、功能设计 |
| 定制能力 | `customization_services` | 尺寸、颜色、工艺、辅料、包装、标签 |
| 研发/设计能力 | `rd_capabilities` | 图案设计、结构设计、打样、CAD/3D |
| 交付能力 | `delivery_cycle`、`capacity` | 7 天打样、45 天量产、年产量 |
| 合规能力 | `certifications` | Oeko-Tex、CP65、行业认证 |
| 信任资产 | `partnerships`、`case_studies` | 大牌合作、项目案例、客户评价 |
| 服务能力 | `after_sales`、`warranty`、`local_service` | 质保、退换、安装、本地服务 |
| 内容/增长能力 | `content_assets`、`marketing_support` | TikTok 脚本、视觉指南、网站素材 |

## 3. 证据强度等级

| 等级 | 定义 | 可否作为核心优势 |
|---|---|---|
| A | 有量化数据、案例、认证、客户名、交付过程证据 | 可以 |
| B | 有明确流程、周期、设备、服务动作 | 可以，但需标注待增强 |
| C | 只有文字描述，没有外部证明 | 只能作为候选 |
| D | 只有口号，如“质量好、服务好、价格低” | 不可作为优势 |

## 4. 同行通用卖点过滤

以下卖点不能直接作为核心差异化：

- 质量好。
- 服务好。
- 价格低。
- 交期快。
- 支持定制。
- 源头工厂。
- 经验丰富。

除非补足：

```text
具体到哪类买家 Job？
快多少、稳多少、低多少？
有什么流程或案例证明？
与同行相比为什么难复制？
```

## 5. 核心优势通过规则

一条能力可以进入 `job_capability_fit` 的条件：

1. 绑定一个 `desired_outcome`。
2. 证据强度为 A 或 B。
3. 不是同行通用卖点，或已被具体化。
4. 能解释买家为什么愿意选择该企业。
