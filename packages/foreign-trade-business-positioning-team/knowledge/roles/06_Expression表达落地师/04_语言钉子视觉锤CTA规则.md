# 04 语言钉子、视觉锤与 CTA 规则

版本：v0.1  
适用角色：`06_Expression表达落地师`

## 1. 语言钉子

语言钉子是定位的压缩表达，不是随意 slogan。

合格语言钉子必须：

1. 承接 `positioning_statement`。
2. 让目标买家快速理解结果。
3. 不使用禁用话术。
4. 能作为首屏或 CTA 前文。
5. 没有语义别扭的组合词。
6. 客户 10 秒内能复述给同事。

示例：

```text
Start small, scale with confidence.
From small-batch validation to flexible production.
Outdoor spaces delivered from design to installation support.
Custom outdoor living sets for contractors, starting from 1 m².
Pergola, outdoor kitchen and furniture, configured by project and shipped together.
```

不合格示例：

```text
Your Shop Drawing. Our 1㎡ Container.
```

原因：有记忆点，但 `1㎡ Container` 把起订面积和集装箱混在一起，语义别扭；可作为内部创意草稿，不作为客户版首屏主语。

## 2. 视觉锤

视觉锤必须让买家更快理解定位。

| 定位路径 | 视觉锤方向 |
|---|---|
| 小单测试供应链 | 样衣/打样流程、从 1 件样衣到批量生产的进度图 |
| 品牌孵化供应链 | 设计稿 -> 样衣 -> 上架 -> 补货 |
| 项目解决方案 | 3D 方案图、项目现场、安装过程 |
| 渠道分销 | 产品组合墙、展厅、热销套装 |

## 3. CTA 类型

| CTA | 适用场景 |
|---|---|
| `Get a product validation plan` | 小单测试 / 品牌孵化 |
| `Request a custom sample` | 定制 / OEM / ODM |
| `Get a project solution quote` | 项目解决方案 |
| `Download product catalog` | 渠道分销 |
| `Ask for compliance documents` | 合规敏感买家 |

## 4. CTA 红线

- CTA 不能只有 `Contact us`。
- CTA 必须承接买家的具体任务。
- CTA 不能要求买家一次性提供过多信息。
- CTA 不能引导到与定位无关的产品页。
