# 03 信任资产与证据展示规则

版本：v0.1  
适用角色：`06_Expression表达落地师`

## 1. 信任资产分类

| 类型 | 页面展示方式 |
|---|---|
| 认证 | 证书图标、适用市场、标准说明 |
| 合作品牌 | 合作 Logo 墙，但必须确认授权和合作类型 |
| 交付过程 | 工厂、打样、质检、包装、出货、安装过程 |
| 项目案例 | 场景图、客户类型、问题、方案、结果 |
| 客户评价 | 截图、评价摘录、复购数据 |
| 本地服务 | 地图、服务点、安装/售后流程 |
| 数据资产 | 打样周期、产能、出口额、复购率、项目数 |

## 2. 展示优先级

```text
与定位直接相关的证据
  > 与目标买家相关的证据
  > 与品类信任相关的证据
  > 泛化公司实力
```

## 3. 授权边界

合作品牌和客户案例必须区分：

```yaml
proof_asset:
  asset_type: "partnership | case | certification | review"
  display_allowed: "yes | no | unknown"
  display_mode: "logo | text | anonymized | internal_only"
```

`display_allowed: unknown` 时，页面 brief 只能写“需确认授权”，不能直接建议展示 Logo。

## 4. 信任状句式

| 证据 | 推荐表达 |
|---|---|
| 7 天打样 | `7-day sampling for faster product validation` |
| 45 天量产 | `45-day bulk production after validation` |
| Oeko-Tex / CP65 | `Compliance-ready for key overseas markets` |
| 项目交付 | `From design drawings to installation support` |
| 120% 配件保障 | `Spare parts prepared for after-sales continuity` |
