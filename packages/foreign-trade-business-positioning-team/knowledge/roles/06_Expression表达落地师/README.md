# 06 Expression 表达落地师：私有知识切片

本目录用于放置 Expression 角色的可交付私有知识。它负责把经营定位转成阿里国际站可执行的首屏、信任资产、7 屏 brief 和 CTA。

## 文件

| 文件 | 用途 |
|---|---|
| `01_表达输入门槛与五成本模型.md` | 定义识别、理解、信任、决策、行动五类成本 |
| `02_阿里国际站7屏Brief模板.md` | 标准 7 屏结构和每屏字段 |
| `03_信任资产与证据展示规则.md` | 认证、案例、合作品牌、交付过程的展示边界 |
| `04_语言钉子视觉锤CTA规则.md` | Slogan、视觉锤和 CTA 的生成规则 |
| `05_案例表达校准.md` | Sweet Home / HOTBLIZZARD 页面表达校准 |
| `06_输出模板与交接格式.md` | 输出标准 YAML，交给 MetricQualityGate 使用 |

## 源引用

`roles/06_Expression表达落地师/knowledge_refs.yaml`

## 角色边界

- Expression 不重新定义定位，只把定位落成表达。
- 信任资产必须来自 SupplyMapping 通过的证据。
- 每屏必须说明降低哪类成本，并能承接买家决策。
