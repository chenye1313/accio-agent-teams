# 02 特劳特心智抢词规则

版本：v0.1  
适用角色：`05_Differentiation差异化定位师`

## 1. 何时使用

当出现以下情况时，优先使用特劳特式定位：

- 成熟品类，买家选择过多。
- 竞争对手表达同质化。
- 企业有一个明确、可证据化的强能力。
- 需要抢占一个简洁心智词。

## 2. 心智词标准

合格心智词必须：

1. 简单。
2. 可被买家理解。
3. 与核心 Job 相关。
4. 有企业能力支撑。
5. 能与竞品形成对比。

## 3. 可用心智词类型

| 类型 | 示例方向 |
|---|---|
| 买家型 | `streetwear startup supply chain` |
| Job 型 | `small-batch testing partner` |
| 能力型 | `7-day sample to scalable production` |
| 场景型 | `commercial outdoor space solution` |
| 风险型 | `low-risk product validation` |

## 4. 抢词流程

```text
优先买家
  -> 核心 Job
  -> 最强能力
  -> 竞品通用说法
  -> 反向压缩成一个词/短语
  -> 测试是否能落到首屏
```

## 5. 竞争对比

输出必须说明：

```yaml
competitor_contrast:
  common_competitor_claims:
    - ""
  our_word:
    - ""
  why_we_can_own_it:
    - ""
  why_not_generic:
    - ""
```

## 6. 红线

不能抢以下词：

- `leader`，除非有市场份额或行业地位证据。
- `best`，除非有第三方评价或数据。
- `one-stop`，除非有完整方案和交付证据。
- `brand incubation`，除非有客户成长案例。

- `process/deep-review/book-indexes-json/B08.json`：B08 U798 强调定位是心智资源占有，品牌是结果。
- `process/deep-review/book-indexes-json/B08.json`：B08 U22 强调选择过多和心智有限会加剧竞争。
- `process/deep-review/book-indexes-json/B08.json`：B08 U462 强调强品牌通常代表一个词或概念。
