# 04 路径选择决策树

版本：v0.1  
适用角色：`05_Differentiation差异化定位师`

## 1. 用途

本文件用于判断定位应走：

- 特劳特式心智抢词。
- 华与华式信任前置。
- 混合路径。

## 2. 决策树

```yaml
path_decision_tree:
  q1_is_category_mature_and_competitive:
    question: "客户所在品类是否成熟、同质化严重、买家选择过多？"
    yes_next: "q2"
    no_next: "q4"

  q2_has_ownable_word:
    question: "是否有一个能被企业证据支撑的简单词/短语？"
    yes: "mind_word"
    no_next: "q3"

  q3_has_trust_heavy_service:
    question: "是否更依赖方案、服务、共创、顾问式信任？"
    yes: "trust_first"
    no: "return_to_supply_mapping"

  q4_is_b2b_consultative_supply_chain:
    question: "企业是否承担客户产品研发、方案设计、市场判断或供应链陪跑？"
    yes: "trust_first"
    no_next: "q5"

  q5_has_job_plus_capability_plus_page_path:
    question: "是否已有清晰 Job、强能力证据，并能转成页面首屏？"
    yes: "hybrid"
    no: "return_to_upstream"
```

## 3. 路径输出要求

| 路径 | 输出重点 |
|---|---|
| `mind_word` | 一个心智词、竞品对比、为什么能占住 |
| `trust_first` | 顾问式身份、首屏信任资产、证明路径 |
| `hybrid` | 心智词 + 信任资产，避免只写口号 |

## 4. 返工规则

| 失败 | 退回 |
|---|---|
| 没有优先买家 | BusinessModel / Intake |
| Job 含产品名 | BuyerJTBD |
| 能力无证据 | SupplyMapping |
| 定位不能落首屏 | Differentiation 自己返工 |
