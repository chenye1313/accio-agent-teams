# 07 MetricQualityGate 经营罗盘质检官：私有知识切片

本目录用于放置 MetricQualityGate 角色的可交付私有知识。它负责把定位动作绑定经营指标，并执行红线质检与返工路由。

| 文件 | 用途 |

| `02_红线质检与评分规则.md` | 六条红线、评分表和通过标准 |
| `03_复盘周期与返工路由.md` | 每日/每周/每月节奏与退回角色 |
| `04_案例回放验收规则.md` | Case 1 / Case 2 回放验收基准 |
| `05_输出模板与交接格式.md` | 输出标准 YAML，交给 Lead 使用 |

## 源引用

`roles/07_MetricQualityGate经营罗盘质检官/knowledge_refs.yaml`

## 角色边界

- MetricQualityGate 不写新定位，只审查定位是否能经营验证。
- 红线失败时必须返工，不能放行。
- `pass_with_gap` 必须显式披露缺口。
