# 03 复盘周期与返工路由

版本：v0.1  
适用角色：`07_MetricQualityGate经营罗盘质检官`

## 1. 三个复盘节奏

| 节奏 | 用途 | 关注 |
|---|---|---|
| 每日 | 异常哨兵 | 首响、消耗、有效询盘异常、高价值询盘漏接 |
| 每周 | 效率穿透 | CTR、CPL、访客转化、询盘转化、跟进质量 |
| 每月 | 战略对账 | ROI、复购、爆款集中度、品类机会、资源配置 |

## 2. 返工路由

| 失败点 | 退回角色 |
|---|---|
| 缺字段 / 资料不足 | Intake |
| 生意模式不清 | BusinessModel |
| Job 不成立 | BuyerJTBD |
| 能力无证据 | SupplyMapping |
| 定位无取舍 | Differentiation |
| 7 屏不能落地 | Expression |
| 指标无法绑定 | MetricQualityGate 自己返工或退回 Differentiation |

## 3. 指标异常与复盘层

| 指标异常 | 优先复盘 |
|---|---|
| 有效询盘率低 | 目标买家、关键词、首屏定位 |
| CTR 低 | 标题、主图、心智词、买家场景 |
| CPL 高 | 投放人群、关键词、目标市场 |
| 访客转化效率低 | 页面结构、信任资产、CTA |
| 询盘转化效率低 | 回复话术、买家 Job、信任状 |
| 报价订单率低 | 差异化、报价逻辑、谈判 |
| 老客户复购比低 | 产品价值、售后、客户分层 |
| 爆款集中度异常 | 产品结构、主推品、风险分散 |
