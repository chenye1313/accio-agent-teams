# 05 案例模式校准

版本：v0.1  
适用角色：`02_BusinessModel生意模式分析师`

## 1. 用途

本文件用于防止 Agent 把案例机械套用到所有客户。

案例只做校准，不做模板复制。

## 2. 案例 1：Sweet Home

来源：`process/cases/case1_sweethome_extract.md`

### 2.1 输入证据

| 证据 | 来源 |
|---|---|
| 重点关注商用客户 | R2-R3 |
| 主产品为凉亭，配套户外家具、橱柜、WPC 地板、围栏 | R4 |
| 支持尺寸、形状、颜色、选配和不同气候生产工艺 | R5 |
| 一站式解决方案、CAD/3D/安装图纸、本地化服务中心 | R7 |
| 信任资产包括品牌墙、好评、交付过程、本地化代理 | R8 |

### 2.2 模式判断

```yaml
primary_business_model: "项目解决方案型"
secondary_business_models:
  - "柔性定制型"
  - "渠道分销型"
unsuitable_models:
  - model: "纯工厂供货型"
    reason: "案例主线不是只卖凉亭产能，而是为酒店、住宅、零售、电商提供场景化方案。"
  - model: "平台 / 服务型生意"
    reason: "有本地化服务和代理，但未形成平台、订阅、抽成或网络效应证据。"
```

### 2.3 关键解释

Sweet Home 的成交理由不是“我们能生产凉亭”，而是：

```text
我们能把户外空间项目从设计、产品组合、安装支持、配件保障到信任资产一起解决。
```

因此它应该优先服务：

- 酒店 / 商业项目。
- 工程承包商。
- 高净值住宅项目。
- 需要方案能力的零售/电商客户。

## 3. 案例 2：HOTBLIZZARD / Shengfeile

来源：`process/cases/case2_hotblizzard_profile.yaml`

### 3.1 输入证据

| 证据 | 字段 |
|---|---|
| 当前定位写明品牌孵化供应链伙伴 | `current_positioning` |
| 目标是帮助街头潮牌创业者从小单测试到稳定量产 | `growth_goal` |
| 当前模式包含品牌孵化供应链、小单定制、柔性制造、ODM/OEM | `current_business_model` |
| 订单类型为小单测试、深度定制、量产补货 | `main_order_type` |
| 7 天打样 + 45 天量产 | `delivery_cycle` |
| 支持面料、工艺、logo、包装、水洗标、挂牌、领标、辅料定制 | `customization_services` |
| 有面料研发、图案设计、辅料工艺、快速打样、数据分析能力 | `rd_capabilities` |
| 优先买家是正在测试新品的街头潮牌创始人 / 小型潮流服饰品牌主 | `priority_buyer` |

### 3.2 模式判断

```yaml
primary_business_model: "品牌孵化供应链型"
secondary_business_models:
  - "小单测试供应链型"
  - "柔性定制型"
  - "OEM / ODM 型"
unsuitable_models:
  - model: "纯工厂供货型"
    reason: "优先买家不是只找低价产能，而是需要从创意、测款到量产的增长路径。"
  - model: "渠道分销型"
    reason: "当前证据不显示其主要服务线下渠道或区域代理。"
  - model: "平台 / 服务型生意"
    reason: "有数据分析与产品策略表达，但未形成平台收入、网络效应或订阅服务证据。"
```

### 3.3 关键解释

HOTBLIZZARD 的主线不是“服装工厂”，而是：

```text
帮助小型潮牌用低风险方式完成产品验证、定制生产和放量补货。
```

它的风险在于：

- “品牌孵化”说法较重，必须继续补产品策略和成功案例证据。
- 如果缺少真实孵化案例，主模式应降级为“小单测试供应链 + 柔性定制”。
- 如果客户只需要成熟大货，品牌孵化表达反而会降低效率。

## 4. 案例反推规则

| 当客户像 Sweet Home | 优先判断 |
|---|---|
| 多场景、多产品组合、工程/空间/安装支持 | 项目解决方案型 |
| 强调案例、信任墙、本地服务、成本降低 | 项目解决方案 + 信任资产 |
| 同时服务商业、住宅、渠道、电商 | 主线必须收敛到一个优先买家 |

| 当客户像 HOTBLIZZARD | 优先判断 |
|---|---|
| 小品牌、测款、低 MOQ、快打样 | 小单测试供应链 |
| 参与设计、趋势、产品策略、内容素材 | 品牌孵化供应链 |
| 多维定制和柔性量产 | 柔性定制 + ODM/OEM |
