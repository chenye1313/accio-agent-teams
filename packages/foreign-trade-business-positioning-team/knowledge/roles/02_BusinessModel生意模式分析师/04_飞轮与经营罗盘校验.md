# 04 飞轮与经营罗盘校验

版本：v0.1  
适用角色：`02_BusinessModel生意模式分析师`

## 1. 为什么要做飞轮校验

经营模式不是标签，而是经营改善路径。

如果一个模式不能解释：

```text
什么动作 -> 改善什么指标 -> 产生什么复购/转介绍/放量 -> 反过来强化什么能力
```

它就只是包装话术，不能进入后续定位。

## 2. 三个校验问题

### Q1：可持续性

增长来自内部能力，还是一次性外部刺激？

- 内部能力：交付更快、案例更多、产品策略更准、客户复购更高。
- 外部刺激：短期广告、低价促销、偶然大客户。

### Q2：核心性

飞轮是否驱动主价值主张？

- 项目解决方案型：项目案例越多，方案能力和信任越强。
- 品牌孵化供应链型：测款反馈越多，趋势判断和爆款能力越强。
- 工厂供货型：产能利用率越高，成本和交期越稳。

### Q3：经营指标绑定

至少绑定 2 个经营罗盘指标：

- 营销效能：CTR、CPL、综合 ROI。
- 团队转化：访客转化效率、询盘转化效率、报价订单率。
- 经营健康：有效询盘率、老客户复购比、爆款集中度。

## 3. 模式飞轮模板

### 3.1 工厂供货型飞轮

```text
稳定交付
  -> 大客户信任
  -> 更稳定补货订单
  -> 产能利用率提高
  -> 成本/交期优势增强
  -> 吸引更多规模采购
```

优先指标：报价订单率、老客户复购比、综合 ROI。

### 3.2 OEM / ODM 型飞轮

```text
研发/打样能力
  -> 客户新品开发成功率提升
  -> 客户把更多 SKU 交给企业
  -> 行业样品库和工艺经验积累
  -> 开发效率提升
  -> 更强 ODM 信任
```

优先指标：询盘转化效率、报价订单率、老客户复购比。

### 3.3 柔性定制 / 小单测试飞轮

```text
低 MOQ + 快打样
  -> 客户更愿意试错
  -> 测款数据与反馈增加
  -> 企业更懂趋势和爆款规律
  -> 推荐方案更准
  -> 小单转量产概率提高
```

优先指标：有效询盘率、询盘转化效率、爆款集中度。

### 3.4 品牌孵化供应链飞轮

```text
产品策略共创
  -> 客户上新成功率提升
  -> 客户复购和补货增加
  -> 企业积累趋势/版型/内容素材
  -> 下一个客户孵化效率更高
  -> 品牌孵化案例增强信任
```

优先指标：有效询盘率、老客户复购比、报价订单率。

### 3.5 项目解决方案飞轮

```text
方案设计 + 产品组合 + 交付服务
  -> 项目风险降低
  -> 形成完整案例和信任资产
  -> 更容易拿到相似项目
  -> 行业场景经验增强
  -> 方案报价能力提升
```

优先指标：访客转化效率、报价订单率、综合 ROI。

### 3.6 渠道分销飞轮

```text
热销产品 + 物料 + 展厅/销售支持
  -> 渠道更容易卖
  -> 渠道补货与扩品类
  -> 企业更了解终端动销
  -> 推荐和备货更准
  -> 渠道粘性提高
```

优先指标：老客户复购比、有效询盘率、综合 ROI。

## 4. 失败模式

| 失败 | 表现 | 处理 |
|---|---|---|
| 广告依赖 | 只有投放能带来询盘 | 标记为外部刺激，不是飞轮 |
| 低价依赖 | 成交靠降价 | 标记毛利风险 |
| 英雄依赖 | 靠老板或个别业务员经验 | 要求流程化证据 |
| 一次性项目 | 项目做完无案例复用 | 要求案例资产化 |
| 小单杂活 | 小单多但不能转量产 | 降低模式评分 |
