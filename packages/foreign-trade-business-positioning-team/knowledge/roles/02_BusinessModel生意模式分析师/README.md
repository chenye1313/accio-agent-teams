# 02 BusinessModel 生意模式分析师：私有知识切片

本目录用于放置 BusinessModel 角色的可交付私有知识。它的任务不是复述“制造商 / 外贸公司 / 品牌方”身份，而是判断客户当前最应该主打哪盘生意。

## 文件

| 文件 | 用途 |
|---|---|
| `01_经营模式分类与判别规则.md` | 定义 9 类经营模式及主/辅模式边界 |
| `02_证据字段与评分规则.md` | 用输入字段给候选模式评分，决定是否阻断 |
| `03_B2B模式选型决策树.md` | 把买家、订单、能力映射到候选模式 |
| `04_飞轮与经营罗盘校验.md` | 检查模式是否能形成经营飞轮并绑定指标 |
| `05_案例模式校准.md` | 用 Sweet Home / HOTBLIZZARD 两个案例校准判断 |
| `06_输出模板与交接格式.md` | 输出标准 YAML，交给 BuyerJTBD 使用 |

## 源引用

`roles/02_BusinessModel生意模式分析师/knowledge_refs.yaml`

## 角色边界

- BusinessModel 只判断经营模式、证据、风险和下游买家分析方向。
- BusinessModel 不写最终定位语、不写页面表达、不做 JTBD 深挖。
- 缺目标买家、订单类型或能力证据时，必须退回 Intake。
